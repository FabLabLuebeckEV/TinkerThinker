Konzepte

InputWas du drückst: button (Taste) · dpad (Steuerkreuz) · axis_pair (Joystick-Achsen analog)
Auslöserpress einmalig beim Drücken  ·  release beim Loslassen  ·  hold dauerhaft solange gehalten
PWMMotorstärke −255 bis 255  ·  positiv = vorwärts  ·  negativ = rückwärts  ·  0 = stop
motor_directSetzt Motor auf festen PWM-Wert. Mit hold: läuft solange Taste gehalten, danach automatisch stop.
drive_pairJoystick (axis_pair) steuert Fahrwerk über zwei Motoren. Ziel gui = Haupt-Fahrwerk (Motor A+B).
servo_nudgeVerschiebt Servo um Delta-Grad pro Druck. Negatives Delta = andere Richtung. Mit hold: kontinuierlich.
servo_toggle_bandWechselt beim Drücken zwischen vordefinierten Winkeln, z.B. 0°→90°→180°→0°.
speed_adjustÄndert den globalen Geschwindigkeitsmultiplikator. Delta +0.1 = 10% mehr Speed.
led_setSetzt LEDs auf eine Farbe. Start = erster LED-Index, Anzahl = wie viele LEDs.
