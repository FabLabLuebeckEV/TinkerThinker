TinkerThinker WebSerial Installer

Flasht das aktuelle Release oder lokale BIN-Dateien direkt in Chrome/Edge und spricht parallel mit der laufenden Firmware ueber eine serielle JSON-Schnittstelle fuer Name, Konfiguration, BT/WiFi-MAC und Live-Logs.

Wenn Web Serial im Datei-Modus blockiert ist, die Datei ueber einen lokalen HTTP-Server oeffnen, zum Beispiel mit python -m http.server im Ordner tools/.

Web Serial wird geprueft Runtime-Serial getrennt Flash bereit

Serielle Ausgabe / Flash-Log

Normale Firmware-Logs und Flash-Ausgaben laufen hier zusammen. Serielle JSON-Antworten werden zusaetzlich rechts strukturiert ausgewertet.

Runtime-Serial

Fuer Monitor, Info, Name setzen und serielle Konfiguration. Nutzt 115200 Baud und die in der Firmware integrierte `TTJSON:`-Antwortstruktur.

Aenderungen an WiFi-/AP-Namen werden gespeichert, greifen aber erst nach Neustart vollstaendig.

Geraeteinfos

BT-MAC, WiFi-MAC, Firmware, Uptime und zuletzt empfangene JSON-Antworten.
